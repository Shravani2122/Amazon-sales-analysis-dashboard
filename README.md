## üìù Project Overview
This is an advanced *Power BI Dashboard* designed to analyze Amazon's sales data. The project provides deep insights into sales performance, geographical trends, and product categories using professional data visualization techniques and complex DAX modeling.

## üöÄ Key Technical Features
* *Lollipop Charts*: Implemented custom Lollipop charts to provide a modern and clean look for comparing product category performance.
* *Advanced DAX & Time Intelligence*: Leveraged complex DAX queries to calculate key metrics such as Year-to-Date (YTD), Month-to-Date (MTD), and Year-over-Year (YoY) sales growth.
* *Custom Date Table*: Created a dedicated and automated Date Table to ensure accurate time-based filtering and seamless Time Intelligence calculations.
* *Interactive Search*: Integrated a functional Search Icon and search bar for quick navigation through large product datasets.
* *Dynamic Multi-KPI Slicers*: 
    * Implemented interactive *Button Slicers* for both *Sales* and *Order Status*.
    * Allows users to toggle between different time periods (7D, 30D, 3M, 6M, 1Y, All).
    * Provides real-time updates to key metrics, enhancing the user's ability to perform granular trend analysis.
